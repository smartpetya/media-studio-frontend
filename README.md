# media-studio-frontend

Фронтенд AI Media Studio. Чистый HTML/JS — один файл. Хостится на GitHub Pages.

## Деплой на GitHub Pages

1. Создай репозиторий `media-studio-frontend` на GitHub
2. Загрузи `index.html` (и `manifest.json` если нужен PWA)
3. Settings → Pages → Source: **Deploy from a branch** → Branch: **main** → Save
4. Через 1-2 минуты сайт будет на `https://smartpetya.github.io/media-studio-frontend/`

## Настройка

После деплоя бэкенда на Render:
1. Открой сайт
2. Нажми ⚙ backend
3. Вставь URL бэкенда (например `https://media-studio-backend-xxxx.onrender.com`)
4. Нажми "Проверить" — увидишь какие API подключены
5. Сохрани
